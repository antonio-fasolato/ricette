## Ingredienti

| Ingredienti                  | Ingredienti             |
| ---------------------------- | ----------------------- |
| **400 g** - Pasta | Yoghurt bianco, panna o latte |
| **1** - Cipolla di tropea | **1 manciata** - Noci |
| **1/2** Dado | Olio evo |
| Sale e pepe | |

## Procedimento

1. Tagliare a fette le cipolle di Tropea
2. Rosolarle con olio, sale, pepe e un po' di dado.
3. Aggiungere una manciata di noci e continuare la cottura.
4. Frullare i tutto
5. Alla fine diluire con un po' di Yogut bianco o Panna o latto
