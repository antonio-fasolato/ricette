## Ingredienti

### Per il ripieno

| Ingredienti                  | Ingredienti             |
| ---------------------------- | ----------------------- |
| **14 oz.** - Latte condensato (397g) | **2 cups** - Zucchero a velo (200g) |
| **4 cups** - Cocco grattugiato (300g) | |

### Per la copertura

| Ingredienti                  | Ingredienti             |
| ---------------------------- | ----------------------- |
| **4 cups** - Cioccolato fondente (680g) | **2 teasp.** - Burro (10g) |

## Procedimento

1. Si mescola il latte con lo zucchero e in seguito il cocco.
2. Si versa in una teglia di circa 33x22cm e si appiattisce
3. Si creano delle file di mandorle nell'impasto e si premono bene con le dita.
4. Si lascia raffreddare in frigo una notte. La mattina dopo si tagliano in rettangoli lasciando due mandorle per ogni ritaglio.
5. Si scioglie a bagnomaria il cioccolato con il burro e si immergono i ritagli.
6. Si lascia raffreddare completamente
